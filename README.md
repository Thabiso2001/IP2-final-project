# 🌾 FarmConnect

A Full-Stack Digital Marketplace Connecting South African Farmers and Buyers.

FarmConnect is a web application developed for the Internet Programming 2 (ITNP200/ITPR200) module at Mangosuthu University of Technology. The system connects farmers directly with buyers through an online marketplace while providing an intelligent farming assistant (FarmBot) for crop pest and disease guidance.

---

## Project Overview

FarmConnect provides a digital marketplace where:

- Farmers can register, manage products, update stock and receive customer orders.
- Buyers can browse products, add items to a shopping cart, place orders and track purchases.
- FarmBot assists farmers by providing crop pest and disease recommendations.

## Installation

### Clone the repository

```bash
git clone https://github.com/Thabiso2001/FarmConnect.git
```

### Navigate into the project

```bash
cd FarmConnect
```

### Install dependencies

```bash
npm install
```

### Configure MySQL

Create the project database and update the database credentials inside the project.

### Run the application

```bash
node server.js
```
